Browser-based ESP flashing

Install Stingray firmware without IDEs or CLI tools.

Plug in the device, open this page in Chrome or Edge, and flash compiled firmware directly over USB with ESP Web Tools.

Compiled binaries only No source code published Static hosting friendly

Before you start

  1. Use Chrome or Microsoft Edge on Windows, macOS, or Linux.
  2. Plug in the ESP32 with a USB data cable.
  3. Close PlatformIO, Arduino Serial Monitor, or any other app that may already own the serial port.

If detection fails

  1. Try a different USB cable or port.
  2. Reconnect the board and refresh the page.
  3. For the XIAO ESP32-C3, hold BOOT, tap RESET, then retry if the board does not appear.
